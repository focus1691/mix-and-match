# mix-and-match
Match two pairs game - psychology themed

## Introduction
A small game made with Phaser 3.

## Installation
`npm install` to install the dev dependencies

## Scripts
- `npm start` to start the express server.
- visit http://localhost:8081.

## The game
Match all the pairs.
